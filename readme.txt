== Description ==


Image Hover effects for WooCommerce products is a collection of pure css3 animations for images. This would allow user to add their WooCommerce products with pure css3 animation effects to increase the interactivity of user to their products. It is a collection of very interesting and attractive hover animations. It is really simple and fast to set up in few seconds. It is really userful for the end user who is not having any programing knowledge.
